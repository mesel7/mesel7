## Hi there 👋

React·Next.js 기반으로 화면뿐 아니라 서비스 전체의 흐름을 설계하는 웹 개발자입니다.  
작은 불편도 놓치지 않고 개선하며, 명확한 소통과 일관된 코드 구조를 중요하게 생각합니다.

React・Next.jsを基盤に、画面だけでなくサービス全体の流れまで設計するWebエンジニアです。  
細かな不便も見逃さず改善し、明確なコミュニケーションと一貫性のあるコード設計を大切にしています。
